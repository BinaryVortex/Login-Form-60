# Login Form

Login Form Built Using HTML, CSS and JavaScript.

This repository contains a simple, clean, and responsive login form built with semantic HTML, modern CSS and light JavaScript for validation and interactivity. It's designed as a small UI component you can reuse or study to learn layout, styling and basic form handling.

## Features

- Simple and modern login UI
- Responsive layout (works on mobile and desktop)
- Basic client-side validation (HTML5 / JavaScript)
- Easy to customize (colors, spacing, fonts)

## Files

- `index.html` - Markup for the login form
- `style.css` - Styles for the UI
- `Screenshot 2024-07-28 000655.png` - Example screenshot shown above

## Usage

1. Clone the repository:

   git clone https://github.com/BinaryVortex/Login-Form-60.git

2. Open `index.html` in your browser (double-click or use Live Server / any static host).

3. Customize the HTML/CSS as needed.

## Customize

- Edit `style.css` to change colors, spacing, and typography.
- Edit `index.html` to add or remove fields.
- Add JavaScript to `index.html` (or a separate file) for more advanced validation or to hook up to a backend.
